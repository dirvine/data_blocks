pub enum BlockIdentifier {
    ImmutableData([u8; 32]),
    StructuredData([u8; 32], DataIdentifier),
    Link(LinkDescriptor),
}

Data identifiers for use in a data Chain. The hash of each data type is available to ensure there is no confusion over the validity of any data presented by this chain

Variants

ImmutableData([u8; 32])

hash is also name of data stored locally

StructuredData([u8; 32], DataIdentifier)

hash name (identity + tag) (stored localy as name in data store)

Link(LinkDescriptor)

Hash of group members' public keys (see LinkDescriptor).
